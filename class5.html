soldering

In this class, what we studied was how to build a basic board.



The PCB board which we had made last class will be used again to install some needed elements to accomplish some function.




The most important thing we need to do is to use the soldering iron to fix the elements on the board. The final purpose is to make the board useful, and it can run the program we write which will be used in different ways.





I’m sorry that I didn’t take photos when I was soldering, but I found some photos on the wechat.











By the way, there are some tips about soldering:



1. We don’t need to make the tin hot immediately. Instead, we can make the pin hot first, and use the hot pin to soldering the tin on it. This way will be useful when we are soldering on the tiny things.

2. After soldiering, we can make the tin hot again, and this action will make the tin form a better shape, and make sure the electric circuit will be worked well.

3. When we make too much tin on it, we should be patient enough. Without more tin, we need to remove the more tin by many times, and make sure it won’t affect the circuit.
