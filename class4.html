pcd board

This class we need to learn how to make a PCB, which will be used in the following classes, and a new machine we will use is called Roland.



The style of the PCB is downloaded from the internet, and it need three steps. First, curve the line Second, draw the hole Last, cut the board




When we prepare for the first step, we need to make a really plane first. This is taught by teacher on the class, and this is important because without this step, the line may have some problems and cause the failure in the future. And another we need to remember is the value of x,y,z. We need to set it before the cutting on the computer, and the z need to be 0 when the tool meet the PCB.
